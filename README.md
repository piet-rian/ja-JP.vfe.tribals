# RimworldXMLModTemplate

This is the template repository for XML-based mods in RIMWORLD.

see. <https://rimworldwiki.com/wiki/Modding_Tutorials/Mod_Folder_Structure>

## MID-SAVE

- 途中導入
  - 問題の有無
- 途中で除去
  - 問題の有無、エラーが発生するかどうか

## CONFLICT

競合が予想されるmodについての記述
